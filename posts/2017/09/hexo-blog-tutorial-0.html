HEOXO 静态博客教程（序）

发表于 2017-09-20 | 分类于 IT | |

教程简介

搭建一个完全属于自己的静态博客，到底需要多少操作？我说的博客，可不是QQ空间这样的平台，在那里一切框架都是搭好的，你只能在局限的空间内去创作。我指的是，每一个页面，每一个链接，每一个功能，甚至是，博客的域名，都可以完全由自己控制的博客。

听上去像是一个庞大的工程？Hexo + Git 的技术会让个人博客建站达到史无前例的便捷。这篇教程会尝试用最精简的步骤和语言，与您分享我在学习搭建 Hexo 博客的过程中的点点滴滴，并且在工具、空间、域名等服务选择上，以免费作为选择供应商的标准。

教程计划

  • 基础部分：博客系统的本地搭建、博客网站的托管。
  • 进阶部分：博客配置和日常运维、Markdown指南
  • 扩展部分：博客插件的使用、域名博客与DNS配置

基础部分的两个工作做完之后，您的博客就算正式上线了，可以被互联网上的任何人用 HTTP / HTTPS 访问到。

进阶部分将为您展示如何按自己的要求去配置网站结构、主题，DIY 博客的方方面面；以及用 Markdown 这样一款优美又简单到爆的语言去书写博文。

扩展部分，如您还想进一步扩展博客的功能，可以使用各类博客插件；以及将博客通过 DNS 指向自己的个性化域名等。
